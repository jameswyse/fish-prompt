# Fish Prompt

My custom fish prompt.

![screenshot]

## Install

With [Fisherman]

```
fisher i jameswyse/fish-prompt
```

## Notes
You'll want to use a powerline enabled font, see [powerline/fonts](https://github.com/powerline/fonts) for more info.
